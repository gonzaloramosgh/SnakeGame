
Juego de la Viborita
Este es un juego de " La viborita " programado en Python utilizando la biblioteca Pygame. En este juego, el jugador controla una serpiente que se mueve por la pantalla y come comida para ganar puntos. Si la serpiente choca contra los bordes de la pantalla o consigo misma, pierde el juego.

Instalación
Descarga el código fuente desde el repositorio.
Instala la biblioteca Pygame utilizando el comando pip install pygame.
Ejecuta el juego con el comando python snake.py.

Controles
Las flechas del teclado controlan el movimiento de la serpiente.
Reglas del Juego
La serpiente crece al comer comida.

Si la serpiente choca contra los bordes de la pantalla o consigo misma, pierde el juego.
El objetivo es conseguir la mayor cantidad de puntos posible antes de perder.

Snake Game
This is a snake game programmed in Python using the Pygame library. In this game, the player controls a snake that moves around the screen and eats food to score points. If the snake collides with the screen edges or with itself, the game is over.

Installation
Download the source code from the repository.
Install the Pygame library using the command pip install pygame.
Run the game using the command python snake.py.

Controls
The arrow keys control the movement of the snake.

Game Rules
The snake grows by eating food.
If the snake collides with the screen edges or with itself, the game is over.
The goal is to score as many points as possible before losing.
